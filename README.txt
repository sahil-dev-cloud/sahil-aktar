# SITE PESSOAL - TRABALHO PRÁTICO Nº5

## ESTRUTURA DO PROJETO

Este projeto consiste em um site pessoal desenvolvido para o Trabalho Prático nº5,
com o objetivo de demonstrar domínio em HTML, CSS e JavaScript.

### REQUISITOS ATENDIDOS:

1. ✓ Estrutura HTML semântica
2. ✓ Múltiplas páginas com navegação entre elas
3. ✓ Folhas de estilo CSS organizadas
4. ✓ Tabelas para apresentação de dados
5. ✓ Imagens e elementos gráficos
6. ✓ Formulário de contato com validação
7. ✓ JavaScript para interatividade
8. ✓ Design responsivo
9. ✓ Código comentado e organizado

### PÁGINAS INCLUÍDAS:

- index.html - Página inicial com apresentação
- sobre.html - Informações pessoais e formação
- portfolio.html - Galeria de projetos
- contato.html - Formulário e informações de contato

### TECNOLOGIAS UTILIZADAS:

- HTML5
- CSS3 (com variáveis CSS e Flexbox/Grid)
- JavaScript (ES6+)
- Google Fonts

### COMO EXECUTAR:

1. Extraia todos os arquivos mantendo a estrutura de pastas
2. Abra o arquivo index.html em um navegador moderno
3. Navegue pelo site usando o menu superior

### OBSERVAÇÕES:

- Substitua as imagens placeholder por suas próprias imagens
- Atualize as informações pessoais nos arquivos HTML
- Modifique as cores no arquivo style.css (variáveis :root)
- Adicione mais projetos conforme necessário
